- Install NodeJS LTS (v20)

```js
   cd PPCinIM-Webinterface
   npm install
   npm run dev
```


- Install flask, flask-cors, python-dotenv
```js
   cd PPCinIM-Webinterface
   flask run  
```


- Run Webinterface
```js
   http://localhost:5173
```

- Start neo4j
```js
   username: neo4j
   password: 12345678
```
